Dockerized Authentication-Protected Todo Web App

This project is a full-stack Todo Web App with authentication and role-based access control, containerized using Docker for seamless deployment.

Features

User authentication (JWT-based)
Create, update, delete, and manage todos
Role-based access control
Responsive UI
Dockerized for easy deployment

Tech Stack

Frontend: React.js, Bootstrap/Material UI
Backend: Node.js, Express.js
Database: PostgreSQL
Authentication: JWT, bcrypt
Containerization: Docker, Docker Compose
